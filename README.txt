・ゲームの説明(操作説明)
  ボタンをクリックして、キューブのラインを回転させて遊ぶルービックキューブです。
  キューブの3面の上部分がA面で、左がB面、右がC面になります。そして、A面の裏側がF面、B面の裏側がD面、C面の裏側がE面になります。
  時計回りAのボタンはA面を正面に見て、右回りでラインを回転します。
  90度右回転AボタンはA面を正面に見て、右回りでキューブ自体を回転させます。(ライン3つを回転させる)
  シャッフルボタンをクリックして、シャッフルできる。
  ボタンにマウスのカーソルを合わせると、回転するラインの色が変わる。
  展開図もキューブと同様にラインが移動する。

※Chrome、Firefoxで動作します。IEなどでは動作しません。
